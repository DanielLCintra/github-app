# Github App

Bem vindo ao projeto easy carros app.


## Preview

![](https://prnt.sc/vmud46)


## Project setup
```
yarn install
```

### Compilar e hot reload para desenvolvimento
```
yarn start
```

### Compilar e minificar para produção
```
yarn build
```

### Executar testes
```
yarn test
```
