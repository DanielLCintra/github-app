# Github App

Bem vindo ao projeto easy carros app.


## Preview

![My image](https://daniellcintra.github.io/images/githubapp.png)


## Project setup
```
yarn install
```

### Compilar e hot reload para desenvolvimento
```
yarn start
```

### Compilar e minificar para produção
```
yarn build
```

### Executar testes
```
yarn test
```
